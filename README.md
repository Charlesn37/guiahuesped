# Guiahuésped - Digital Guest Guide

Guiahuésped es una solución de guía digital de alto rendimiento para rentas vacacionales, diseñada específicamente para el contexto de México y mercados internacionales. 

Este repositorio contiene la implementación base estática (frontend) del sistema CarAle (Riviera Nayarit).

## 🚀 El Producto

Guiahuésped elimina la fricción entre el host y el huésped mediante una interfaz 100% estática, carga ultrarrápida y diseño centrado en la accesibilidad (WCAG 2.1 AA).

### Características principales:
- **Zero-Latency:** Carga instantánea sin dependencias de base de datos en el cliente.
- **Config-Driven:** Personalización total de la propiedad mediante `config.js`.
- **Vanilla Stack:** Pureza técnica (HTML/CSS/JS) para máxima compatibilidad.
